# Regression Test

## Workflow

- Reproduce or explain the bug.
- Find root cause.
- Add a failing regression test if possible.
- Make the minimal fix.
- Run the regression test.
- Check the regression checklist.
- Do not rewrite unrelated code.
- Report remaining risk if a regression test cannot be added.
