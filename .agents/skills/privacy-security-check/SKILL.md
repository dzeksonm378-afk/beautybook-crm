# Privacy Security Check

## Check

- Auth bypass.
- Missing authorization.
- IDOR.
- RLS gaps.
- Leaked client data.
- Leaked internal IDs.
- Sensitive logs.
- Unsafe errors.
- Unsafe exports.
- Unsafe uploads.
- Token/session issues.
- Role permission bypass.
- No secrets, cookies, session IDs, Supabase keys or personal data in code/logs.
