# Feature Implementation

## Workflow

- Find existing code paths.
- Understand current flow before editing.
- Plan a small implementation.
- Implement the minimal diff.
- Add or update tests for affected behavior.
- Run relevant checks.
- Browser verify if UI changed.
- Review CRM business rules.
- Review RLS, privacy and security.
- Review regression risks.
- Finish with summary, files, checks, browser result, business/security review and remaining risks.
