# Project Bootstrap

## Workflow

- Inspect project structure.
- Check Next.js App Router.
- Check TypeScript.
- Check Tailwind.
- Check shadcn/ui.
- Check Supabase setup.
- Check env examples.
- Check package scripts.
- Do not create secrets.
- Do not install dependencies without approval.
- Report missing setup with exact recommended commands.
