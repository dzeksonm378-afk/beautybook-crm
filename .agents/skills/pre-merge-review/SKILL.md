# Pre Merge Review

## Review

- Correctness.
- CRM business rules.
- Appointment/calendar regressions.
- Missing tests.
- RLS/security.
- UI states.
- Error handling.
- Dependency changes.
- Unrelated changes.
- Debug logs.
- TODOs.
- Tests, lint, typecheck, build and browser smoke.

## Verdict

- merge readiness: yes/no;
- blocking issues;
- non-blocking issues;
- checks run;
- recommended next step.
