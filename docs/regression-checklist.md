# Regression Checklist

Flows:
- registration;
- login;
- logout;
- session restore;
- protected routes;
- create salon after registration;
- client list;
- create client;
- edit client;
- delete client;
- client search;
- client profile;
- service list;
- create/edit service;
- master list;
- create/edit master;
- dashboard;
- appointments list;
- create appointment;
- edit appointment;
- reschedule appointment;
- cancel appointment;
- complete appointment;
- mark no-show;
- appointment conflict prevention;
- analytics;
- settings;
- mobile layout;
- navigation;
- error handling.

For each touched flow, check happy path, invalid input, empty state, error state, authorization, RLS isolation where applicable, and mobile layout if UI is involved.
